text-indent設為正數適合文章首行縮排兩個字寬
程式碼可以這樣寫:
#textindent-p {
font-size: 20px;
text-indent: 2em;
background-color: #ccc;
padding: 1em;
}

#text-indent設為負數適合首字是一個圖示
程式碼可以這樣寫:
textindent-p {
font-size: 20px;
text-indent: -1em;
background-color: #ccc;
padding: 1em 1em 1em 2em;
}
